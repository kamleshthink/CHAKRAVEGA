# Chakravega Technologies

A refined corporate website built for Chakravega Technologies, designed to present the company's engineering, innovation, and digital transformation capabilities with a polished modern experience.

## Overview

This repository hosts the frontend for Chakravega Technologies, a high-performance corporate website developed with Next.js, TypeScript, and Tailwind CSS. The site communicates the company’s enterprise-grade product portfolio, sector expertise, leadership vision, talent pipeline, and contact infrastructure.

## Key Highlights

- Professional landing experience with a full-screen hero, animated visual sections, and clean corporate typography
- Business units and product showcases for technology offerings like JhariaWatch and PragyaTek
- Industry-focused messaging tailored to mining, government, manufacturing, energy, and transportation
- Leadership and careers sections for brand credibility and talent acquisition
- Contact form and inquiry workflows for client outreach and partnership requests
- Responsive design with subtle motion, visual hierarchy, and accessible navigation

## Technology Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Local Development

```bash
npm install
npm run dev
```

Open the site at `http://localhost:3000` (or the next available port if 3000 is occupied).

## Production Build

```bash
npm run build
npm run start
```

## Repository Structure

- `src/app/` — Application shell and page entry points
- `src/components/sections/` — Page sections and modular UI components
- `public/images/` — Brand assets and site imagery
- `next.config.mjs` — Next.js configuration
- `tailwind.config.ts` — Tailwind CSS configuration

## Suggested Enhancements

- Add a serverless contact API route for secure message delivery
- Integrate a CMS solution for dynamic content management
- Add analytics and SEO metadata for production readiness
- Deploy to a managed platform such as Vercel or Netlify

## Notes

This project is built as a corporate marketing website with a strong emphasis on professional branding, enterprise messaging, and modern web performance.
